# Polter Website UI Kit

Marketing landing page. Single column, paper-warm light background, ghost as the only character.

## Files

- `index.html` — the landing page itself.
- `Hero.jsx` — full-bleed dark hero with the floating ghost.
- `VoiceSamples.jsx` — three things the ghost might say.
- `PrivacyNote.jsx` — the "what we never see" section.
- `Footer.jsx`
